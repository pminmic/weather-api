# Weather Display on LCD Using Raspberry Pi and an API

This project involves displaying the weather conditions of a specific city on an LCD screen connected to a Raspberry Pi. The weather data is fetched using an API and displayed in real-time on the LCD. This API is from [el-tiempo.net](https://www.el-tiempo.net/api) with provides data for Spain.
